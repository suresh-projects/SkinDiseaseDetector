# Skin Disease Detector (Android App)

An Android-based skin disease detection application using Deep Learning and TensorFlow Lite.

## 📌 Project Overview
This application predicts skin diseases using smartphone images. 
The model is trained using EfficientNetB0 on the HAM10000 dataset and deployed using TensorFlow Lite.

## 🚀 Features
- Image selection from gallery
- Real-time skin disease prediction
- Confidence score display
- Nearby dermatologists location view
- Clean and user-friendly UI

## 🧠 Technologies Used
- Java (Android)
- TensorFlow Lite
- EfficientNetB0
- OpenStreetMap
- Android Studio

## 📊 Dataset
HAM10000 - Human Against Machine Skin Lesion Dataset

## 📱 How It Works
1. User selects or captures a skin image.
2. Image is preprocessed.
3. TensorFlow Lite model predicts the disease class.
4. Result and confidence score are displayed.
